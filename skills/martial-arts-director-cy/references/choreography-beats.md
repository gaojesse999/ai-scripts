# 武指段子骨架库（choreography-beats）

按「宫格规格 × 武术体系/兵器」预置参考用招式骨架。每条骨架按编号列出招式名 + 一句话动作要点（朝向 / 重心 / 发力部位）。

> ⚠️ **使用规则（反硬编码）**：本文是**招式词典**，不是**答案模板**。
> - 选用骨架时，**必须根据用户的人物性格、场景调性、影视参考、兴趣点**做实质性调整（替换、改写、重组、增删）。
> - 每次生成至少调整 **1/3 招式**，不要原样套用任何骨架。
> - 同一题材连续生成两次，招式列表应有明显差异（不同的招式名 / 不同的姿态分配 / 不同的强调点）。
> - **起势 / 收势招**必须从 SKILL.md Step 2 的"仪式感动作池"中根据角色调性动态选择，不要默认"双手负后正面站姿"。

**使用方式**：Step 2 阶段，根据用户的「宫格规格 + 武术体系 + 是否带兵器」选最接近的骨架作为起点，再按上方规则做实质性定制。

**通用四段结构**：
```
起势段（1-2 招）：抱拳礼 / 立定亮相 / 收马
进攻段（占 1/3）：直拳 / 扫腿 / 肘膝 / 摔投
防御反击段（占 1/3）：闪身 / 格挡 / 反击 / 控制
收势段（1-2 招）：定格 / 收马 / 收兵礼
```

## E. 兵器与徒手骨架的扩展约定

- **兵器**：每一格的招式描述必须含兵器名词与运动轨迹（"刀光横扫""棍头戳击""枪口指向"）
- **双人对练**：每招写两个角色（A 主动 / B 被动），并标注两人空间关系（距离 / 朝向）。本表只列单人骨架，对练版需在 Step 2 临时改写
- **腾空 / 跌落动作**：在 16 格里最多 2 招（容易跨格漂移），且要选定格清晰的瞬间（"空中下劈定格"而非"翻滚中段"）
- **重复招式**：同一海报内不重复同名招式；如要变化（左/右版本），写明"左 / 右"区分

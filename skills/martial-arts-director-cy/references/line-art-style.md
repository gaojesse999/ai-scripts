# 线稿风格脊柱（line-art-style）

本 skill 的图片**默认风格 = 高清写实线稿图（line art / 精密线描稿）**。所有 Step 3 多宫格海报、单格高清图，演武主体一律用「黑色线条 + 干净白底 + 排线表达体积明暗」的写实线稿呈现，**不上色、不做照片级写实渲染**——线稿需精确还原 8K IMAX 院线真人实拍质感所需的全部结构信息，使后续可由 AI 从线稿还原为电影级写实风格。

> ⚠️ **本文件是「风格脊柱」，不是「答案模板」**：固定块（🟢）原则保持不变、按语义嵌入；需要更新块（🔴）必须由 agent 根据用户的人物锚定卡 / 招式列表 / 场景动态填充；可调整块（🟡）按宫格规格与用户需求调整。同一题材两次生成，需要更新块的具体描述应有可观察差异。

---

## 一、线稿技术规范（🟢固定，每次必嵌）

最终输出为高清写实线稿图（line art / 精密线描稿）。以纯净、精准的线条呈现人物：黑色线条 + 干净白色背景，高分辨率清晰线稿，线条流畅肯定、粗细有节奏变化，外轮廓线肯定、结构线与转折线准确，体积与明暗通过排线 / 影线（hatching、cross-hatching）表达，不做色彩填充（或仅极少量灰度过渡），保持纯线稿质感。

————（以下为"目标还原风格"参考：用于决定线稿应精确刻画的结构与质感，并非最终上色）————

超写实电影级人物，8K IMAX 院线真人实拍质感，高预算电影剧照效果。非动画、非插画、非二次元风格，以真人演员方式呈现真实影视作品中的人物形象。采用 Arri Alexa 65 大画幅电影摄影机拍摄风格，全画幅电影摄影，真实镜头光学特性，自然焦外虚化，轻微真实镜头畸变。线稿需精确还原达到 8K 超高清的结构精度，保留高光与阴影边界（以明暗交界线 / 影线表示），完整呈现可由 AI 还原写实风格所需的全部结构信息。采用基于真实物理结构的线条逻辑，真实光照下的明暗关系（以线条疏密表达），真实材质的纹理走向，真实摄影物理逻辑下的透视与体积。避免 CG 廉价感，避免塑料 AI 感，避免游戏角色感。

## 二、写实人体与材质线条逻辑（🟢固定，每次必嵌）

人物符合真实人体物理规律：真实人体比例，自然身体结构，真实肌肉和骨骼比例。线稿精确刻画皮肤的真实微观结构——毛孔、细微皮肤纹理、自然皱褶、皮肤油脂高光（以留白 / 高光线表示）；表现物理准确的次表面散射区域过渡（以渐变排线暗示），体现真实人体皮肤质感。头发符合真实物理规律——自然发丝结构（逐缕线条）、真实光泽（以高光留白表示）、轻微受环境影响产生运动。服装符合真实布料物理规律——自然褶皱（清晰褶皱结构线）、重量感、重力下垂效果、真实布料纹理走向。整体光影：电影级布光关系，柔和主光，均衡补光，环境反射光；以线条疏密形成自然且具有戏剧张力的明暗结构。

## 三、人物身份 / 构图 / 背景（🔴需要更新，由 agent 按本次任务填充）

这一块是**本 skill 与线稿脊柱的接口**——把固定的线稿规范套到当前这套武术分镜上。填充来源：

- **人物身份 / 角色设定**：写成一段自成一段的角色块（开头「演武者为同一位 …」结尾「全程保持同一人物」），便于后续角色替换。线稿仍要精确刻画这位演武者真实人体结构与服装布料结构。**命名分流（v10.2）**：
  - **已提供角色参考图** → 角色块**直接引用「参考图挂载名 / 引用名」**（用户给上传参考图起的标签，如 `@女主`、`@角色1`，**不是参考图人物的真实角色名 / IP 名**），让 i2i 工具按那张参考图来画；不要再堆泛化外貌（如"一位古典神话风格的女战士"）；挂载名后最多补 1-2 个动作 / 兵器 / 气质锚点。
  - **未提供角色参考图** → 用 Step 1「人物锚定卡」的泛化描述（年龄 + 性别 + 气质 + 武术身份 + 服装材质颜色款式 + 三个面部 / 体型锚点），不出现任何名字。
- **构图（多宫格语境下的适配）**：N 宫格分镜排版，每一格演武者全身居中、不裁切手脚头顶；机位随招式动态变化（部分格平视 / 部分格略微低机位仰拍体现威严力量 / 部分格俯拍体现全局）；人物比例真实准确、姿态自然、非摆拍（是运动中的瞬间，不是僵硬定格）。
- **背景空间深度**：用 Step 1「场景」字段，以简洁结构线 / 透视线表示真实空间深度，空气透视，电影级景深关系（远景线条更轻更疏、近景线条更重更密）；背景不喧宾夺主，主体线条始终最清晰。

填充示例（仅供语感，不照抄）——
- **无参考图**：「演武者为同一位约 25 岁东方古典气质女性武者，身形修长、肩背挺拔、眉目英气，穿真实布料质感的窄袖劲装与束腰，全程保持同一人物；每格全身居中、略微低机位电影视角，姿态为运动中的真实瞬间；背景以简洁透视结构线表现纵深古战场，远景线条更轻更疏。」
- **有参考图**：「演武者为同一位{参考图挂载名，如 @女主}，全程保持同一人物，身形与持械按参考图；每格全身居中、略微低机位电影视角，姿态为运动中的真实瞬间；背景以简洁透视结构线表现纵深古战场，远景线条更轻更疏。」（{参考图挂载名} 是上传参考图的标签，不是参考图人物的真实角色名）

## 四、画面排除项（🟢固定，每次必嵌）

不要：卡通、动漫、二次元线稿、插画上色、游戏 CG、廉价特效、塑料材质。不要：色彩填充、照片级上色、水印、Logo。画面要求：线条轮廓清晰、主体明确、低视觉噪点、低细节堆积、低装饰密度、线稿干净统一。

> ⚠️ **关于文字 / UI 与本 skill 多宫格注释的协调**：线稿脊柱原则上排除画面内文字 / 字幕 / UI。但本 skill 的多宫格海报需要保留**功能性工作单注释**（大号阿拉伯数字编号 / 中文招式标题 / 简洁中文要点 / 动作方向箭头 / 重心轨迹线 / 格子分隔线）——这是武指工作单的刚需，且 Step 4 视频 prompt 已显式排除它们不进入视频。因此规则调整为：**演武主体 + 背景一律纯线稿、无任何上色与照片级渲染**；画面唯一允许的"非线稿元素"是上述功能性工作单注释（细线条、轻量字体，不抢主体）；除此之外严禁任何水印 / Logo / 装饰文字。若用户明确要"纯净无注释线稿"，则连工作单注释一并去掉，只留纯线稿分镜。

## 五、画幅与输出目标（🟡可调整）

- **画幅比例**：多宫格海报默认 1:1 或 4:3（容纳 4×4 / 3×3 网格）；若用户要单格 / 单人竖屏分镜，可用 9:21 竖屏。
- **输出目标**：4K 以上高清写实线稿图，可供 AI 从线稿还原为上述电影级写实风格。

## 六、嵌入方式（agent 操作指引）

1. Step 3 组装多宫格海报 prompt 时，把「一、线稿技术规范」+「二、写实人体与材质线条逻辑」+「四、画面排除项」按语义融进 poster-template 的风格定语 / 服装 / 背景 / 整体效果各处（不必逐字照搬，但线稿核心关键词必须出现：黑色线条、干净白底、排线 / 影线表达体积明暗、不上色、可还原写实）。
2. 「三、人物身份 / 构图 / 背景」用本次人物锚定卡 + 场景 + 招式列表实际填充。
3. 「五、画幅与输出目标」放进给用户的使用建议（不进 prompt 正文）。
4. 单格高清图（Step 4 的【A. 单格高清图提示词】）同样用线稿脊柱：演武主体纯线稿、可还原写实。
5. Step 4 视频 prompt：参考图是线稿分镜，需在「参考图阅读协议」里加一句「线稿还原声明」——告诉视频模型把线稿结构还原渲染为上述 8K IMAX 院线真人实拍写实风格再动起来（线稿已编码全部写实结构信息），其余 UI 注释仍按原规则排除。

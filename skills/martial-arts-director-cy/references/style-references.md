# 影视武指风格参考库（style-references）

10 个标志性武指作品/团队的风格特征，供 Step 1 命中"影视参考"字段后，在 Step 3 的【调性定语】【整体效果参考】【风格关键词】三处自动注入对应词汇。

每条结构：**调性定语 / 镜头偏好 / 节奏偏好 / 推荐配套词**。

---

## 1. 袁和平 ×《卧虎藏龙》《一代宗师》《黑客帝国》
- 调性定语：写意武侠 / 飘逸轻灵 / 衣袂翻飞
- 镜头偏好：长镜头跟拍、低角度仰拍、慢动作配威亚
- 节奏偏好：起承转合明显、招式之间有"呼吸"、注重亮相定格
- 推荐配套词：白衣、竹林、屋瓦、剑气、踏雪无痕、一袭长袍

## 2. 甄子丹 ×《叶问》《杀破狼》《导火线》
- 调性定语：港式硬桥硬马 / 写实硬派 / 招招到肉
- 镜头偏好：贴身中近景、稳定固定机位、清晰可见招式细节
- 节奏偏好：连消带打节奏密集、寸拳爆发短促、定格瞬间硬朗
- 推荐配套词：唐装、咏春日字拳、肘膝近战、叶问范、对打不留情

## 3. 成龙 × Stunt Comedy（《警察故事》《醉拳 II》《尖峰时刻》）
- 调性定语：动作喜剧 / 环境互动 / 道具化招
- 镜头偏好：长镜头 oner、广角全景、跟拍奔跑
- 节奏偏好：流畅一气呵成、招式之间有滑稽过渡、节奏快慢交错
- 推荐配套词：椅子、桌子、梯子、长棍、街头巷尾、笨拙中藏精确

## 4. John Wick × 87eleven（David Leitch / Chad Stahelski）
- 调性定语：gun-fu / 冷峻精准 / 暴力美学 / 长镜头 oner
- 镜头偏好：低角度过肩跟拍、长镜头不切、暗光高对比
- 节奏偏好：连贯紧凑无停顿、近身战 + 射击交替、面无表情
- 推荐配套词：黑色西装、夜店霓虹、双手枪、战术换弹、过肩摔接抢射、巴西柔术

## 5. 朴赞郁 ×《老男孩》走廊长镜头
- 调性定语：长镜头压迫 / 写实疼痛感 / 单挑群斗
- 镜头偏好：单镜头横移、固定走廊视角、地形利用
- 节奏偏好：缓慢压迫、一击一打、累积式疲劳感
- 推荐配套词：走廊、铁锤、灰色衬衫、贴墙战、侧身躲避、连续撞击

## 6. 加雷斯·埃文斯 ×《突袭》《突袭 2》
- 调性定语：写实硬派 / Pencak Silat / 高强度近战
- 镜头偏好：中近景、手持镜头、快速剪辑
- 节奏偏好：每招都有反应动作、缠斗到底、不留情
- 推荐配套词：印尼武术、刀战、楼梯、廊道、警服、贴身缠斗

## 7. 昆汀·塔伦蒂诺 ×《杀死比尔》
- 调性定语：高速凌厉 / 后现代美学 / 血浆飞溅或纯净抽象
- 镜头偏好：剪影对决、慢镜定格、单色背景突出主体
- 节奏偏好：定型瞬间多、剪辑跳切、配乐 80s rock
- 推荐配套词：日本刀、黄色连体服、雪地、单色剪影、东京新娘、群斗

## 8. Matrix 三部曲（Yuen Wo Ping × Wachowski）
- 调性定语：赛博 / 子弹时间 / 咏春 + 柔道融合
- 镜头偏好：环绕镜头、子弹时间慢镜、低角度俯仰
- 节奏偏好：超慢动作 + 正常速度交替、空中悬浮定格
- 推荐配套词：黑色风衣、墨镜、长走廊、绿色矩阵光、子弹时间、空中倒挂

## 9. Atomic Blonde（David Leitch）
- 调性定语：机能格斗 / 真实疼痛感 / 长镜头 oner
- 镜头偏好：跟拍长镜头、楼梯间、霓虹反光
- 节奏偏好：用尽手边一切道具、累积式疲惫、女性主角
- 推荐配套词：金发、机能服、霓虹蓝、楼梯间、玻璃碎、皮带绳

## 10. 李小龙 × 截拳道时代（《精武门》《龙争虎斗》《死亡游戏》）
- 调性定语：古典动作片 / 截拳道 / 一对一单挑
- 镜头偏好：固定中景、定型亮相清晰、节奏感配合嘶吼
- 节奏偏好：寸止 + 爆发交替、注重单招亮相、怒目嘶吼
- 推荐配套词：黄黑战衣、双截棍、寸拳、肌肉线条、怒目嘶吼、东方哲学

## 风格组合 / 互斥提示

- **同一份海报只用一种调性**：写意武侠 vs 暴力美学不能混；港式硬桥 vs 长镜头压迫不能混
- **可组合**：调性 + 时代背景（如"港式硬桥 + 民国"= 叶问感；"gun-fu + 雨夜霓虹"= John Wick 感；"写意武侠 + 竹林晨雾"= 卧虎藏龙感）
- **风格 → 服装锁定**：选风格后，服装方向也基本被锁定（gun-fu = 黑西装 / 写意武侠 = 白衣长袍 / 截拳道 = 黄黑战衣 / Stunt Comedy = 街头便装）
- **风格 → 场景锁定**：写意武侠 → 竹林屋瓦；gun-fu → 夜店走廊；长镜头压迫 → 走廊楼梯；高速凌厉 → 雪地或单色背景

# 多宫格海报提示词主模板（poster-template）

三个变体：**基础版（徒手）/ 兵器版 / 对打版**。

变量用 `{{xxx}}` 标注，Step 3 阶段必须全部替换为具体描述，**不能留任何 `{{}}` 字面**。

输出时必须是**纯中文文本**，不带 markdown 装饰符号、不带编号符号、不带英文引号。

> ⚠️ **填充原则（反硬编码）**：模板里所有 `{{xxx}}` 不是简单字符串替换，而是**根据用户输入定制描述**。
> - `{{人物三个面部锚点}}` 不是套固定模板（如永远写"剑眉星目、面色清冷、长发束起"），而是根据角色（年龄、性别、性格、气质、影视参考）写具体且差异化的特征。
> - `{{招式 N}}` 必须按 SKILL.md Step 2 的"动态生成原则"动态产出，不要从骨架照抄；起势 / 收势招必须从仪式感动作池中根据角色调性选择。
> - `{{服装}}` `{{场景}}` 等都要根据用户输入和影视参考做差异化描述，不要套用任何固定句式。
> - 每次生成必须使整段提示词与上次同题材生成有可观察的差异（人物锚点 / 服装细节 / 招式名 / 动作要点 / 景别分配 等至少 1-2 项明显不同）。

> ⚠️ **线稿风格优先（默认渲染风格，必读 `references/line-art-style.md`）**：本 skill 图片**默认渲染为「高清写实线稿图 / line art / 精密线描稿」**——演武主体与背景一律用「黑色线条 + 干净白底 + 排线 / 影线（hatching、cross-hatching）表达体积明暗」呈现，**不上色、不做照片级写实渲染、不做 CG / 二次元 / 插画**；但线稿要精确还原 8K IMAX 院线真人实拍质感所需的全部真实人体结构、皮肤纹理、布料褶皱、明暗交界结构信息，使后续可由 AI 从线稿还原为电影级写实风格。
>
> - `{{人物长相风格}}`、`{{服装}}` 仍按用户人物锚定卡填充为具象演武者描述（年龄 + 性别 + 气质 + 武术身份 + 服装材质颜色款式 + 三锚点），但落到画面上是**这位真实人物的写实线稿**（线条精确刻画其真实人体比例、肌肉骨骼、皮肤微观结构、布料物理褶皱）。
> - 线稿技术规范 / 写实人体与材质线条逻辑 / 画面排除项三块固定脊柱，按 `references/line-art-style.md` 融进下方模板的【风格】【服装】【背景】【整体效果】各处，线稿核心关键词必须出现：黑色线条、干净白底、排线 / 影线表达体积明暗、不上色、可还原写实。
> - **武器 / 场景 / 调性 / 武术体系 / 镜头语言 / 强化关键词 / 多宫格工作单注释等其他字段全部正常保留**——线稿只是"渲染风格"，动作精度 / 武器形态 / 镜头节奏 / 对手剪影 / 编号箭头注释一律不变。
> - 仅当用户明确要做版权规避或动作 retarget 时，才把人物换成「纯白色 3D 白模人形 / 线框模型」等抽象人形（此时仍用线稿呈现）。

- 画面为 {{宫格规格}} 的 {{N}} 宫格排版，每一格对应一个连续武术动作步骤，编号 1-{{N}}。
- 整体不是普通教学图，而是带有 {{调性定语}} 的武术动作分镜线稿海报，兼具电影武指感与海报视觉冲击力。

- 每一格都要有：大号阿拉伯数字步骤编号（明显放大、置于格子角落）、中文招式标题（黑体加粗）、简洁中文要点说明（10-20 字、轻量字体）、动作方向箭头或重心轨迹线（细线条、不抢主体）——这些是功能性工作单注释，是画面中唯一允许的非线稿元素，除此之外不要任何水印 / Logo / 装饰文字。

## 变体 B｜兵器版

在变体 A 基础上，**新增**「兵器锚定块」插在「服装」之后：

```
- 兵器为 {{兵器名称}}：{{材质 + 颜色 + 长度 + 关键细节}}，每一格都必须出现，且兵器在格间保持同一外观；兵器在动作中扮演核心角色，{{兵器运动特征：刀光弧线 / 棍头方向 / 枪口指向 / 鞭身波浪}}。
```

并把【动作连贯句】中的招式名替换为含兵器动作的招式（如"劈棍""拔刀斩""双枪交叉射击"）。

招式列表里至少 80% 的格子要有兵器主导动作；徒手过渡招式（如"持械礼""收兵礼"）只占首尾。

## 变体 C｜对打版（双人对练）

把【演武者】块替换为：

```
- 画面有两位演武者：A {{A 人物完整描述}}，B {{B 人物完整描述}}。两人风格 {{相同 / 对立}}，全程保持各自外观不漂移。每格中两人空间关系要明确（距离、朝向、谁主动谁被动），动作要构成攻防对应。
```

把【动作连贯句】改成 A B 互动序列：

```
- 动作顺序为 A 与 B 的攻防对练：{{招式 1：A 起势 / B 起势}}、{{招式 2：A 进攻 / B 反应}}、{{招式 3：B 反击 / A 防御}}、……、{{招式 N：定格亮相}}。
```

每一格的招式名要写成"A 直拳 + B 闪身格挡"这种攻防对的形式。

## 字数参考与排查

主提示词目标长度：**700-1050 中文字符**。

- 不到 700 字：调性 / 服装 / 场景 / 招式名词中至少有一项写得太空，或者强化关键词（动态/法术/手部/对手）漏掉了
- 超过 1050 字：招式描述句太啰嗦，把每招的"一句话要点"压到 5-8 字内

**自动排查 checklist**：

- [ ] `{{}}` 字面已全部替换
- [ ] 招式名按编号顺序全部出现
- [ ] 调性定语只用一种、不混用
- [ ] 人物三锚点已写出（如"短马尾、英气剑眉、修长比例"）
- [ ] 服装含材质 + 颜色 + 款式
- [ ] 兵器（如有）外观描述完整、且在动作连贯句中至少出现 2 次
- [ ] 背景明确、氛围光明确、不喧宾夺主
- [ ] 每一格的差异化要求写明（朝向 / 重心 / 张开度 / 发力部位）
- [ ] 字数在 600-900 中文字符
- [ ] 整段无 markdown 装饰、无英文引号、无参数符号
